# Mini CRM API

Мини-CRM на FastAPI с авторизацией по JWT, PostgreSQL, тестами и Docker.

---

## Технологии

- Python 3.12
- FastAPI
- PostgreSQL 15
- SQLAlchemy 2.0
- Pydantic v2
- Alembic
- Docker / Docker Compose
- Pytest
- GitHub Actions (CI)

---
